## 1. Yêu cầu môi trường
- Node.js: phiên bản khuyến nghị 18.x  
- Java JDK: 17  
- Android Studio 
- Đã cấu hình biến môi trường:
  - ANDROID_HOME
  - JAVA_HOME
  - PATH có platform-tools (adb)

## 2. Cài đặt thư viện
Thư mục `node_modules/` không được đẩy lên GitHub nên cần chạy `npm install` trước khi build.

## 3. Chạy ứng dụng Android
npx react-native run-android
